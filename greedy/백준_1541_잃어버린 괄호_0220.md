### 풀이 방법
1. '-' 를 기준으로 나눈다.
2. '+'를 기준으로 나눈후, 더한다.
3. 2번에서 더한 값들을 빼준다.

### 코드
```python
import sys 

arr = sys.stdin.readline().split('-')

# 첫번째 숫자는 더해야하니까, 먼저 계산.
answer = sum(list(map(int, arr[0].split('+'))))

for i in arr[1:]:
    answer -= sum(list(map(int, i.split('+'))))

print(answer)
```
